## 🚀 Como Usar

### **1. Instalação Local**
```bash
# Clone o repositório
git clone [url-do-repositorio]

# Navegue para a pasta
cd adilsonchitunda

# Abra o arquivo index.html no navegador
# Ou use um servidor local
python -m http.server 8000
```

### **2. Personalização**
- **Cores**: Edite as variáveis CSS em `styless.css`
- **Conteúdo**: Modifique o HTML em `index.html` e `contacto.html`
- **Funcionalidades**: Ajuste o JavaScript nos arquivos `.js`
- **Imagens**: Substitua as imagens placeholder por imagens reais

### **3. Deploy**
- **Netlify**: Arraste a pasta para o dashboard
- **Vercel**: Conecte o repositório Git
- **Firebase Hosting**: Use `firebase deploy`
- **GitHub Pages**: Ative nas configurações do repositório

## 🔧 Configuração Firebase

### **1. Criar Projeto Firebase**
- Acesse [console.firebase.google.com](https://console.firebase.google.com)
- Crie um novo projeto
- Ative Firestore Database
- Configure Authentication

### **2. Atualizar Configuração**
```javascript
// firebase-config.js
const firebaseConfig = {
  apiKey: "sua-api-key",
  authDomain: "seu-projeto.firebaseapp.com",
  projectId: "seu-projeto-id",
  storageBucket: "seu-projeto.appspot.com",
  messagingSenderId: "seu-sender-id",
  appId: "seu-app-id"
};
```

### **3. Regras Firestore**
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
```

## 💬 Chat Widget

### **Funcionalidades**
- **Contornos Vermelhos** - Botão e cabeçalho do chat
- **Mensagens Coloridas** - Usuário (fundo preto) e robô (fundo amarelo)
- **Animação de Digitação** - Indicador visual de digitação do robô
- **Respostas Automáticas** - Sistema de respostas pré-definidas

### **Como Usar**
1. O widget de chat aparece automaticamente na página de contato.
2. Clique no botão vermelho no canto inferior direito para abrir o chat.
3. Digite sua mensagem e envie.
4. O robô responderá com uma animação de digitação.

### **Personalização**
- **Cores**: Edite as variáveis no arquivo `chat-ai.js`
- **Respostas**: Adicione ou modifique respostas no objeto `responses` em `chat-ai.js`
